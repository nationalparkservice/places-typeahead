Places Typeahead

Autocomplete search for the Places system. Supports searching through all point of interest types in Places. Also supports submitting new places and editing existing places.

Use this page to configure and preview the plugin and generate code you can use to embed it in an NPS Bootstrap form. The "Preview" and "Code" tabs update as changes to the plugin configuration are made on the "Configure" tab.

Current filter configuration: All types in all parks.

Use this form to configure the Places Typeahead plugin. Once you've made the configuration changes, click "Apply Configuration" to update the "Preview" and "Code" tabs.

Load NPS Bootstrap, jQuery, and the CSS/JavaScript for the plugin:

<link href="http://www.nps.gov/lib/bootstrap/3.3.2/css/nps-bootstrap.min.css" rel="stylesheet">
<link href="http://www.nps.gov/lib/bootstrap/plugins/places-typeahead/{{ version }}/places-typeahead.min.css" rel="stylesheet">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
<script src="http://www.nps.gov/lib/bootstrap/plugins/places-typeahead/{{ version }}/places-typeahead.min.js"></script>

Then insert an input and instantiate the plugin:

<div class="form-group">
  <label for="places-typeahead">Start typing the name of a place to search for a point of interest.</label>
  <input class="form-control places-typeahead" id="places-typeahead" placeholder="e.g., abo visitor center, old faithful, macarthur meadow..." type="text">
</div>
